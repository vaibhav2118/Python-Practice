## 🎯 Project Objective
The goal is simple: **Write Python code every day.** 
Whether it's a small script, a complex algorithm, or exploring a new library, this repository tracks the journey toward becoming a Python pro.

## 🚀 Repository Structure
- `scripts/`: Implementation of daily logic.
- `challenges/`: Solutions to competitive programming problems.
- `projects/`: Small mini-projects built during practice.

## 🛠️ How to Use
1. **Clone the repo:**
   ```bash

   ```
2. **Navigate to the folder:**
   ```bash
   cd PythonDaily
   ```
3. **Run a script:**
   ```bash
   python scripts/filename.py
   ```
